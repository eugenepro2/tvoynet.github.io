Стоимость и объём услуг при подключении

В многоквартирных жилых домах по технологии FTTH для физических лиц.

    В подключение входит:

  • затяжка кабеля по квартире абонента требуемой длины до компьютера;
  • настройка компьютера для работы в сети и Интернет
  • установка и настройка ПО необходимого для работы в локальной сети и Интернет;
  • присвоение одного семизначного лицевого счета;
  • присвоение одного локального IP адреса;
  • краткий инструктаж по работе в сети.

Условия подключения

Любые затраты на организацию канала связи и его эксплуатацию, связанные с нетипичным подключением, считаются отдельно и оговаривается дополнительными соглашениями, в случае необходимости.

Сетевая карта не входит в стоимость подключения. Хотя она присутствует почти во всех современныхкомпьютерах, при необходимости ее можно установить дополнительно. Стоимость сетевой карты 110грн., с установкой 160 грн.

  • стоимость подключения - 0,01 грн;
  • при подключении, абонентом вносится аванс на счет Интернета в размере ежемесячного платежа;
  • по умолчанию подключен тариф "Твой безлим 60", до момента подключения можно заказать
  • более высокую скорость, после подключения абонент самостоятельно может заказать на следующий месяц любой из доступных тарифов.
